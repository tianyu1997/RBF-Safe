# Support

RBF-Safe is currently maintained on a best-effort basis.

- Use [GitHub Issues](https://github.com/tianyu1997/RBF-Safe/issues) for
  reproducible bugs and scoped feature requests.
- Use the documentation index in [README.md](README.md) for installation,
  input formats, API behavior, compatibility, and safety limitations.
- Include the RBF-Safe version, platform, compiler or Python version, minimal
  input, expected behavior, actual behavior, and relevant logs in bug reports.
- Do not post suspected false-positive certificates or security-sensitive
  parser failures publicly; follow [SECURITY.md](SECURITY.md).

The project does not currently provide commercial support, deployment safety
approval, or guaranteed response times.
